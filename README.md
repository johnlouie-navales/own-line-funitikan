# OWN-LINE FUNITIKAN

### Enhancing the Low Engagement of Grade 7 Faith in Filipino Fables Through Web-Based Interactive Approach

**OWN-LINE FUNITIKAN** is an interactive web application designed to help Grade 7 students engage with Filipino Fables through gamification. The application combines reading comprehension with interactive word search puzzles to reinforce learning.

## üöÄ Features

* **Interactive Fables:** 3 curated Filipino fables with moral lessons.
* **Gamified Assessment:** Integrated Word Search (Hanap Salita) games for every story.
* **Offline-First:** Runs completely offline without needing an internet connection or a backend server.
* **Progress Tracking:** Automatically saves student progress using LocalStorage.
* **Strict Validation:** Ensures students identify words correctly (Context Clues).

* **LocalStorage:** For persisting data without a database.

## üì¶ How to Run (Offline Mode)

Since this project is designed for classroom environments with limited internet access:

1.  Download or copy the project folder to your computer.
2.  Locate the `index.html` file inside the folder.
3.  Double-click `index.html` to open it in any modern web browser (Chrome, Edge, Firefox).
4.  No installation or server (XAMPP/WAMP) required.
